# Drive Simulator 3D

![Preview](img/preview.jpg)

**Drive Simulator 3D** is an interactive web-based driving simulator built with **Three.js**. It allows users to view and drive a 3D model directly in the browser smoothly and with intuitive controls.

## 🚀 Features

- **3D Graphics in the Browser:** Real-time rendering using Three.js.
- **Interactive Controls:** Move the vehicle using the `WASD` keys or the arrow keys.
- **Dynamic Camera:** Drag to navigate around the vehicle and use the mouse wheel (scroll) to zoom.
- **Immersive Interface:** Visual control panel, model loading indicator, and direct links to the creator's profile.
- **Responsive Design:** Adapted for different screen sizes.

## 🛠️ Technologies Used

- **HTML5 & CSS3**
- **JavaScript (ES6 Modules)**
- **Three.js** (v0.160.0)

## 🎮 Controls

| Action | Key / Mouse |
| :--- | :--- |
| **Accelerate / Forward** | `W` or `Up Arrow` |
| **Brake / Backward** | `S` or `Down Arrow` |
| **Turn Left** | `A` or `Left Arrow` |
| **Turn Right** | `D` or `Right Arrow` |
| **Rotate Camera** | `Left Click + Drag` |
| **Zoom** | `Mouse Wheel` |
